Design Patterns
Repository
Data Layer
Abstracts DB operations. TenantRepository enforces workspace_id on every query — impossible to accidentally leak data.
Service Layer
Business Logic
Encapsulates business rules. Services call repositories, never write raw SQL. Testable in isolation.
Provider / Strategy
AI Layer
Swap AI vendors via env var. 7 providers implement AIProvider ABC. Zero vendor-specific business logic.
Factory
Provider Creation
Single point of provider instantiation. Reads AI_PROVIDER env var, returns correct provider instance.
Observer / Events
Analytics, Audit
22 event types flow through Redis Streams. Decouples side effects from core logic.
Outbox
Webhook Delivery
Write to outbox in same transaction as business logic. Background processor delivers with retry + circuit breaker.
Dependency Injection
FastAPI Endpoints
Services injected via Depends(). Enables easy testing with mock dependencies.
Feature Flags
Plan Gating
20 feature flags enforce plan boundaries at code level. Not just UI — API rejects unauthorized features.
