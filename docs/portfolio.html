Provider Abstraction
Swap AI vendors with one environment variable — zero vendor-specific business logic.
# .env
AI_PROVIDER=anthropic   # Was "openai" — that's it

# app/ai/factory.py
def create_provider() -> AIProvider:
    provider = os.getenv("AI_PROVIDER", "openai")
    return PROVIDER_MAP[provider]()
Tenant Isolation
Enforced at the repository layer — impossible to accidentally leak data between workspaces.
# app/repositories/base.py
class TenantRepository(BaseRepository):
    def list_by_workspace(self, workspace_id: UUID):
        return self.db.query(self.model) \
            .filter(self.model.workspace_id == workspace_id) \
            .all()

Design Patterns
Interview-Worthy Architecture
Every pattern chosen to demonstrate system design thinking and production engineering.
Repository Pattern
Data Layer
TenantRepository enforces workspace_id on every query. Developers can't accidentally write SELECT * FROM documents.
Provider / Strategy
AI Layer
7 providers implement AIProvider ABC. Factory reads AI_PROVIDER env var. Zero vendor-specific business logic.
Outbox Pattern
Webhook Delivery
Write to outbox in same DB transaction as business logic. Background processor delivers with retry + circuit breaker.
Event-Driven
Cross-Cutting
22 event types via Redis Streams. Document upload triggers embedding, analytics, and notification consumers independently.
Feature Flags
Plan Gating
20 feature flags enforce plan boundaries at the API level. Free users can't trigger expensive AI operations.
Dependency Injection
FastAPI Endpoints
Services injected via Depends(). Enables easy testing with mock dependencies and loose coupling.
